Vår Java Mocca er en kaffeblanding hvor mesteparten av kaffen er bærtørket. Dette gir en en søt og fruktig smak. Java Mocca er en av de mest klassiske kaffeblandingene som finnes og besto tradisjonelt av Java fra Indonesia og Mocca fra Yemen.

I vår Java Mocca har vi valgt å legge til litt kaffe fra andre kafferegioner for å kunne få en enda mer komplett smak.
